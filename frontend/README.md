# ATS Frontend — Angular 20

This is the Angular 20 frontend for the **Altuwijri Ticket System**.

For full project documentation, setup instructions, and deployment guide see the [root README](../README.md).

## Development

```powershell
npm install
npm start          # http://localhost:4200
```

## Production Build

```powershell
& "C:\Program Files\nodejs\node.exe" ".\node_modules\@angular\cli\bin\ng.js" build --configuration=production
# Output: dist/frontend/browser/
```
